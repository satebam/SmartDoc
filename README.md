## Installation

1. Clone the repository

git clone [your-repository-url]

3. Create a virtual environment:

python -m venv venv
source venv/bin/activate

3. Install required packages:

pip install -r requirements.txt

4. Start the Streamlit app:

streamlit run app.py

5. Access the application in your web browser (typically http://localhost:8501)

6. Upload one or more documents (supported formats: PDF, TXT)

7. Enter your processing prompt in the text area

8. Select output preference (Display/Save to file)

9. Click "Process" to start the analysis
